Выводы для BODRO
Рыночные возможности
Что рынок делает плохо — и как BODRO занимает это пространство.
01 / ВКУС
Вкус — главный пустой сегмент
Средний вкус рынка — 4.2/10. Никто не делает еду, которую хочется снова. Палыч берёт 409 ₽ за 2/10. Даже лучшие (ВкусВилл свёкла — 8/10) — исключение. BODRO со стандартом ≥7 на каждое блюдо — в совершенно свободном сегменте.
Ключевая ставка
02 / БЕЛОК
Рынок не решил протеиновую задачу
Медиана белка на порцию по конкурентам — 7.8г. Это вдвое меньше дневной нормы на приём пищи для активного человека. Лучший — Палыч Курица (20.4г), но вкус 2/10. BODRO SMART с ≥25г белка и вкусом ≥7/10 — уникальное предложение.
BODRO SMART
03 / СВЁКЛА С ФЕТОЙ
ВкусВилл показал: свёкла с фетой — хит
Оценка 8/10, единственный «отличный» в дегустации. Это прямой сигнал: запустить версию BODRO на этот SKU в первые два месяца. Яркий цвет, понятные ингредиенты, фотогеничность — идеальный контент для Telegram.
Быстрый запуск
04 / ЧИСТОТА
Никто не держит обещание «чистого состава»
ЙУМИ: сыр-заменитель. FitBox: ЗОЖ-слоган, дешёвые макароны. ПриЕм: пюре с майонезом. Среднее по рынку — 4.0/10 индекса чистоты. BODRO с обещанием «только то, что ты можешь назвать» и фактическим соответствием — сильное бренд-УТП.
Бренд-позиция
05 / ЦЕНОВОЕ ОКНО
250–320 ₽ — незаполненное окно
Самокат (177–239 ₽): бюджет + посредственно. Палыч/ВкусВилл (319–438 ₽): дорого при среднем качестве. BODRO в диапазоне 250–320 ₽ с высоким вкусом занимает зону максимального value — место без сильного игрока.
Ценовая стратегия
06 / УПАКОВКА
Никто не сделал упаковку, которую хочется сфотографировать
ЙУМИ пробует — тёмная коробка хороша. Но внутри алюминий и заменители. Нет игрока с последовательной упаковкой: красивая снаружи + красивая еда внутри. Белая минималистичная крышка BODRO — незанятое пространство.
Дифференциация
Итоговый вердикт

Кто делает хорошо

ВкусВилл — лучшая упаковочная система и отдельные блюда (свёкла с фетой). Лидер по чистоте состава. Но нестабильность между SKU разрушает доверие.

ЙУМИ — единственный, кто вложился в shelf appeal упаковки. Но наполнение не дотягивает.

Кто провалился

Палыч — системный провал по всем метрикам. 409 ₽ за 2/10 вкус и 2/10 состав. Худшее value на рынке.

FitBox — маркетинг ЗОЖ без подтверждения в тарелке. Дешёвые ингредиенты убивают доверие к категории.

Что означает для BODRO

Рынок готовой еды Самары технически не занят в сегменте «вкусно + честно + разумная цена + достаточно белка».

Среднее по рынку: вкус 4.2, чистота 4.0. Планка входа низкая.

Рынок задал планку 4.2/10 на вкус и 4.0/10 на чистоту состава. BODRO с целевыми стандартами ≥7.5 вкус и ≥8.5 чистота входит в практически незанятый сегмент — при ценовом окне 250–320 ₽, которое не занял ни один игрок. Первый приоритет: свёкла с фетой + курица гриль с гарниром — два доказанных хита без конкурента.
